#  VERIETH | DIGITAL IDENTITY

> **Status:** `ONLINE`  
> **Version:** `1.0.0`  
> **Concept:** `Brutalist / Minimal / Kinetic`

Одностраничное портфолио (цифровая визитка), разработанное с упором на визуальную агрессию, типографику и микро-взаимодействия. Сайт отрицает стандартные шаблоны, используя эстетику **Swiss Style** и **Digital Brutalism**.

## Особенности

*   **Zero Frameworks:** Чистый HTML5, CSS3 и Vanilla JS. Никаких лишних зависимостей.
*   **Bento Grid Layout:** Адаптивная сетка, вдохновленная модульными интерфейсами.
*   **Kinetic Typography:** Огромные заголовки, бегущие строки (Marquee) и игра с начертаниями шрифтов (`Syne` + `Space Grotesk`).
*   **Micro-interactions:**
    *   Кастомный курсор с эффектом инверсии цвета (`mix-blend-mode: difference`).
    *   Эффект цифрового шума (Grain overlay).
    *   "Toast" уведомления при копировании Email в буфер обмена.
*   **Responsive:** Полная адаптация под мобильные устройства без потери стиля.

## Технический стек

*   **Core:** Semantic HTML5
*   **Styling:** Modern CSS (Grid, Flexbox, CSS Variables, Animations)
*   **Scripting:** ES6 JavaScript (DOM manipulation, Clipboard API)
*   **Fonts:** Google Fonts (Syne, Space Grotesk, Space Mono)
